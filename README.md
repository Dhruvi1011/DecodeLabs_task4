# Image Recognition using MobileNetV2 🖼️🤖

This is my fourth project completed during the DecodeLabs AI Internship.

## Project Overview

This project uses the pre-trained MobileNetV2 deep learning model to recognize objects in images.

The model analyzes an input image and predicts the most likely objects present along with confidence scores.

## Features

* Pre-trained MobileNetV2 model
* Image classification using Deep Learning
* Top 3 prediction results
* Confidence percentage for each prediction
* Uses ImageNet-trained weights

## Technologies Used

* Python
* TensorFlow / Keras
* NumPy
* MobileNetV2

## How to Run

```bash
python image_recognition.py
```

## Sample Output

The model displays the top 3 predicted objects along with their confidence scores.
